# Bakemalai 🧁

> Sweets to life — A premium Bangladeshi bakery experience.

A static bakery website with interactive cart system and Netlify Functions backend.

## Features

- **Home Page** — Brand showcase with hero section
- **Shop** — Product catalog with pricing
- **Gallery** — Photo gallery of baked goods
- **About Us** — Story and team info
- **Interactive Cart** — Add/remove items, calculate totals
- **AI Chatbot** — Netlify Functions backend (Google APIs + Sharp for image processing)

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | HTML + CSS + JS |
| Serverless | Netlify Functions |
| APIs | Google APIs |
| Image Processing | Sharp |
| CLI | Netlify CLI |
